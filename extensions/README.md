# extensions

Extensions for [pi.dev](https://pi.dev/docs/latest/extension://pi.dev/docs/latest/extensions)

## install
- copy the file project local in `.pi/extensions/` or globally in `~/.pi/agent/extensions` 

## use
- To test an extension without clutter: `pi --name "extension test"  --no-tools --no-extensions --no-skills --no-context-files --no-themes --no-prompt-templates --thinking off "ping" -e <extension.ts>`
